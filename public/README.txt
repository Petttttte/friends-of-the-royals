PUBLIC ASSETS FOLDER
====================

Place your assets here:

/public/logo.svg          - Primary Friends of the Royals logo (SVG)
/public/logo-white.svg    - White version for dark backgrounds
/public/logo-icon.svg     - Hexagon emblem only
/public/favicon.ico       - Browser tab icon (32x32)
/public/og-image.jpg      - Social share image (1200x630)

EVENT PHOTOS:
/public/images/events/    - Place event photography here
/public/images/team/      - Team and volunteer photos
/public/images/projects/  - Turf field and project photos

SPONSOR LOGOS:
/public/sponsors/         - Place sponsor logo files here
                            Name them: sponsor-name.png or .svg

Note: SVG files are preferred for logos. PNG files should be
high resolution (at least 2x).
